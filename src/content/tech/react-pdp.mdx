## 담당 업무

기존 HTML/SCSS 기반 PDP를 React로 전환하는 과정에서, CSS Modules + SCSS 환경을 도입하고 주요 UI 컴포넌트를 신규 개발했습니다. 아마존 스토어 묶음할인 모듈, 하단 고정 버튼, AI 리뷰 요약 UI, CDN CSS 내재화 등 다양한 모듈을 구현했습니다.

## 주요 성과

- **묶음할인 상품 모듈** — 상품 카드, 수평 스크롤 리스트, 장바구니 추천 레이어 등 3개 컴포넌트 개발
- **하단 고정 버튼** — 장바구니/선물하기 버튼 리팩토링, Figma 디자인 스펙과 SCSS 정합성 보정
- **AI 리뷰 요약 UI** — 그라데이션 애니메이션 구현, Lottie 애니메이션 연동
- **CDN CSS 내재화** — CDN에서 로드하던 외부 CSS를 Critical/Rest 2분할 번들링으로 전환하여 초기 로딩 성능 최적화
