## 담당 업무

모바일웹 정적 리소스 프로젝트의 전체 SCSS 파일을 node-sass에서 Dart Sass로 마이그레이션했습니다. CSS 산출물 변경을 최소화하면서 단계적으로 전환한 대규모 작업입니다.

## 주요 성과

- 전체 SCSS 파일을 폴더별 단계적으로 전환하며 **CSS 산출물 변경을 최소화**
- 빌드 설정 수정(foundation 경로, SVG sprite 최신화), watch 기능 이슈 해결
- 완료 후 정적 리소스 저장소에도 동일 작업 적용하여 **팀 전체 SCSS 환경 통일**
