## 배경

11번가 UI 개발팀에서는 오랫동안 Node Sass(LibSass)를 사용해왔습니다. LibSass가 deprecated된 이후에도 레거시 빌드 환경 때문에 전환을 미뤄왔는데, 2025년 중반 Dart Sass로의 전면 전환을 결정했습니다.

전환 대상 파일은 총 **2,384개**. 단순 문법 차이뿐 아니라 `/deep/`, `::v-deep` 등 deprecated된 선택자들도 대거 포함되어 있었습니다.

## 어떻게 했나

Cursor AI와 Claude를 활용해 패턴 기반 자동 변환 스크립트를 작성했습니다.

```bash
# 주요 변환 패턴
/deep/ → :deep()
@import → @use / @forward (필요한 경우)
```

AI가 패턴을 인식하고 변환 스크립트를 작성했고, 저는 예외 케이스를 검증하는 역할을 했습니다.

## 결과

- 3주 만에 전체 파일 전환 완료
- 빌드 에러 없이 배포 성공
- CI 파이프라인 내 Dart Sass 컴파일 속도 향상

## 느낀 점

대규모 반복 작업에서 AI 코딩 도구의 가치가 명확하게 드러났습니다. 패턴이 명확한 작업은 AI에게 맡기고, 사람은 예외와 검증에 집중하는 방식이 효과적이었습니다.
