## 담당 업무

나의11번가를 시작으로 11번가 서비스의 반응형 웹 전환을 검토하고 로드맵을 수립했습니다. 반응형 적용 방식 비교, font 대응 검토, SVG Sprite 환경 구성, HTML Include 모듈화 등 기반 기술을 정립했습니다.

## 주요 성과

- **반응형 적용 로드맵 수립** — CSS Media Query / JS Framework / Class 분기 방식 비교, AS-IS 분석
- **반응형 font 대응 검토** — px/em/rem 전환 함수 설계, 타이포그래피 가이드 작성
- **SVG Sprite 환경 구성** — 서비스별 Image Request 축소 플랜, 가이드 작성
- **HTML Include 모듈화** — gulp-file-include 적용으로 공통 요소 모듈화
