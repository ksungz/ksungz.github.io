## 담당 업무

Cursor AI를 업무 전반에 도입하여 개발 생산성을 향상시키고, 팀 차원의 AI 도구 체계를 구축했습니다. 코드 리뷰, 문서화, 커밋, PR 생성, 구현 계획 수립 등 반복적인 작업을 자동화하고, 8개 저장소에 AI PR Review Agent를 일괄 적용했습니다.

## 주요 성과

- **14개 커스텀 스킬** 직접 설계 — 커밋, PR, 코드리뷰, 위키, Jira, 구현계획, QA, Slack 연동 등 워크플로우 전반 자동화
- **15개 커스텀 규칙** 설정 — Git 워크플로우, 코드 품질, 협업, 인프라 일관 규칙 체계
- **AI PR Review Agent** — Bitbucket Pipelines에 통합, 8개 저장소 일괄 적용, On-Demand 방식으로 전환
- **ui-scripts 패키지** — UI개발팀 전용 Cursor AI 설정 관리/배포 패키지 (사내 패키지 레지스트리 배포, 저장소 일괄 동기화)
- **MCP 연동** — Bitbucket, Jira, Confluence, Slack 통합 업무 환경 구현

## 커스텀 스킬 체계 (14개)

| 분류 | 스킬 |
|------|------|
| Git/PR | 커밋 메시지 자동 생성 (husky 규칙 연동), PR 디스크립션 자동 생성 및 원스텝 PR 생성/업데이트 |
| 코드 리뷰 | SCSS/마크업/접근성 체크리스트 기반 코드 리뷰 자동화 |
| 문서/이슈 | Confluence 위키 읽기/작성/수정 자동화, Jira 이슈 생성 자동화, 작업시간 기록 자동화 |
| 계획/검증 | 스펙 기반 구현 계획 수립 및 영향범위 분석, QA 검증 자동화 (사이드이펙트/런타임 에러 검사) |
| 커뮤니케이션 | Slack 메시지 읽기, 대화/허들 요약, 주간 다이제스트 자동 생성 |
| 인프라/도구 | Sentry 에러 모니터링 설치 자동화, 에이전트 컨텍스트 관리, 스킬 생성 메타 도구 |

## 커스텀 규칙 체계 (15개)

| 분류 | 규칙 |
|------|------|
| Git 워크플로우 | push 검증, merge 충돌 해결, husky 에러 대응, 브랜치 네이밍 |
| 코드 품질 | 코드 주석 원칙, 코드 리뷰 원칙 |
| 협업 | Jira 티켓 생성 확인, Bitbucket PR 워크플로우, Confluence 위키 규칙 |
| 인프라 | Bitbucket Pipelines 규칙, CLI 도구 사용 규칙 |

## AI 활용 실무 적용 사례

| 영역 | 활용 내용 |
|------|---------|
| 코드 리뷰 | PR diff 기반 SCSS/HTML 마크업 전문 리뷰, Critical/Warning/Suggestion 분류 |
| PR·커밋 | 브랜치 diff 분석 → 디스크립션/메시지 자동 생성 → 원스텝 실행 |
| 문서 작성 | Jira/기획서 기반 Confluence 위키 자동 생성, Storybook 검수 가이드 자동화 |
| 구현 계획 | 기획서 분석 → Phase별 구현 계획 수립, 영향범위 분석 |
| 접근성 점검 | ARIA, 시맨틱 HTML, 대체 텍스트, 키보드 내비게이션 자동 점검 |
