AI 코딩 도구 덕에 속도는 늘었는데, 어느 순간 우리 코드베이스가 낯설어졌다. Cursor가 만들어낸 파일이 어디에 붙어 있는지, Copilot이 추가한 클래스가 기존 구조와 어떻게 연결되는지 — PR 리뷰 때 그냥 넘어가는 게 점점 많아졌다.

CodeBoarding은 그 불안감에 정직하게 답한다. 정적 분석과 LLM 추론을 결합해 코드베이스를 탐색 가능한 아키텍처 다이어그램으로 바꾼다. Stars 2k, Fork 176. VS Code 확장과 GitHub Action을 지원해서 PR마다 구조 변화를 자동으로 잡아낼 수 있다. 멀티 서브앱으로 복잡해진 프로젝트에 붙여보고 싶다.
