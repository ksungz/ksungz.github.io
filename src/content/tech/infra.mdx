## 담당 업무

UI개발팀이 관리하는 서버 및 배포 환경의 인프라 전반을 담당했습니다. Node.js 버전 업그레이드, Docker 이미지 관리, Bitbucket Pipelines 파이프라인 구성, 서버 대개체 대응, 저장소 이관 등 인프라 운영 전반을 수행했습니다.

## 주요 성과

- **정적 리소스 저장소 통합 구축** (2021~2022) — PC/MW 분리 저장소를 하나로 통합, Gulp 빌드 프로세스 설계
- **디자인 시스템 저장소 초기 구축** (2023~2024) — Bitbucket Pipelines prod 배포 환경 구성, 브랜치 전략 정립
- **Node.js 12 → 18/20 전환** — Docker 이미지 업데이트, 7개 저장소 일괄 적용
- **서버 대개체 대응** — CentOS EOS로 인한 서버 교체 후 배포 검증 및 싱크
- **저장소 이관 후 파이프라인 구성** — Bitbucket Server → Cloud 이관에 따른 yml 파이프라인 초기 구성
- **서버 부팅 자동실행 스크립트 설정** — 4대 서버 부팅 시 자동 실행 스크립트 설정
